Statiscope URL Encoder

URL Encoder for Statiscope

Links that loads data
This JavaScript program makes links to Statiscope that cause it to start with data loaded. You enter what you would have written in Statiscope's Data/Location box in the text box below, then click Make link, and a link to Statiscope will appear. You can include this link in your own web-pages.

Examples:

  • A list: 5,3.5,9,3.5
  • A file on Internet: foton.dat, 2nd column
